# Player Classes
PlayerClasses — это плагин для сервера Minecraft (Paper), который добавляет систему классов с уникальными способностями, уровнями и очками опыта. Игроки могут выбирать класс при входе в игру и получать различные бонусы в зависимости от действий

## Описание классов
Помимо своих условий для каждого класса, все игроки получают очки за выполнение достижений. За каждые тысячу очков повышается уровень игрока, что увеличивает бонусы для каждого из классов

### Crafter:
**Бонус при крафте**: С небольшим шансом при крафте используемые ресурсы, находящиеся в верстаке, сохраняются

**Получение очков**: Крафтеры получают очки за разблокировку новых рецептов для верстака и при использовании наковальни

**Уровни**: В зависимости от уровня, увеличивается шанс сохранения ресурсов при крафте. 1% на первом уровне, 5% на втором, 10% на третьем и более высоких

### Hobbit
**Уменьшенный размер**: Хитбокс игрока уменьшается на 34%, что позволяет проходить на шифте в один блок

**Бонус от еды**: Хоббитам еда даёт больше сытости и насыщения, что позволяет меньше её тратить

**Получение очков**: Хоббиты получают очки за каждый пройденный на шифте блок

**Уровни**: В зависимости от уровня, увеличивается бонус от еды. +1 очко сытости и насыщения на первом уровне, +2 на втором, +4 на третьем и более высоких

### Runner:
**Увеличенная скорость бега**: При беге на игрока накладывается эффект скорости

**Уменьшенный урон от падения**: При падении с высоты часть потерянного здоровья восстанавливается сразу же

**Получение очков**: Раннеры получают очки за каждый пройденный блок без бега

**Уровни**: В зависимости от уровня, увеличивается скорость бега и уменьшается урон от падения. Скорость II и на 10% меньше урона на первом уровне, Скорость III и на 25% меньше урона на втором, Скорость IV и на 50% меньше урона на третьем и более высоких

## Команды
- **/class info [class]** - Посмотреть информацию об указанном классе; при отсутствии аргумента выводится общая информация
- **/class set <player> <class>** - Установить указанному игроку указанный класс; доступна только админам для использования на всех и тем, у кого ещё не установлен класс, для использования на себе
- **/class show [player]** - Посмотреть класс, установленный указанному игроку; возможность указать аргумент присутствует только у админов, при отсутствии аргумента показывается класс игрока, использовавшего команду
- **/lvl <add|set> <player> <level>** - Увеличить / изменить уровень игрока на указанное число; только для админов
- **/lvl show [player]** - Показать уровень игрока; возможность указать аргумент присутствует только у админов, при отсутствии аргумента показывается класс игрока, использовавшего команду
- **/points <add|set> <player> <points> - Увеличить / изменить счёт игрока на указанное число; только для админов
- **/points show <player> - Показать счёт игрока; только для админов

## Кастомизация
Плагин можно настроить под свой сервер, используя *config.yml* и *messages.yml*

## Установка
- Скачайте *classes-0.1-shaded.jar*
- Поместите его в папку *plugins/* в директории своего сервера
- Перезапустите сервер
- При необходимости, отредактируйте *config.yml* в *plugins/Player_classes/*, и перезапустите сервер для принятия изменений
